# 📚 SumoNoteBook - AI Knowledge Library

An ever-growing encyclopedia that organizes your notes, documents, and research materials automatically.

## 📂 Directory Structure

```
c:\butler_sumo\library\SumoNoteBook\
├── raw\                    # Raw materials (you put files here)
│   ├── article1.md
│   ├── code.py
│   └── processed\          # Processed files
│
├── Sumo_wiki\              # Organized knowledge base (maintained by AI)
│   ├── index.md            # Main index
│   ├── health_report.md    # Health check report
│   ├── concepts\            # Concept notes
│   ├── summaries\           # Summaries
│   ├── backlinks\           # Backlinks
│   └── qa\                  # Q&A records
│
└── scripts\                # Scripts
    ├── daily_organizer.py   # Daily organization (4:12 AM)
    ├── health_check.py      # Health check (5:21 AM)
    └── query_interface.py  # Query interface
```

## 🚀 Quick Start

### 1. Add Raw Materials
Drop documents into the `raw` folder. The AI will automatically organize them.

### 2. Run Organization Manually
```bash
python c:/butler_sumo/library/SumoNoteBook/scripts/daily_organizer.py
```

### 3. Run Health Check
```bash
python c:/butler_sumo/library/SumoNoteBook/scripts/health_check.py
```

### 4. Query the Knowledge Base
```bash
python c:/butler_sumo/library/SumoNoteBook/scripts/query_interface.py "Your question"
```

### 5. Generate Presentations
```python
from scripts.query_interface import generate_marp_slides
slides = generate_marp_slides("Python Programming")
```

## ⏰ Automation

Set up Windows Task Scheduler or cron:

| Time | Task | Script |
|------|------|--------|
| 4:12 AM | Daily Organization | daily_organizer.py |
| 5:21 AM | Health Check | health_check.py |

## 🔧 Key Features

- **Obsidian Compatible**: All notes are standard Markdown
- **Bidirectional Links**: Each note has backlinks to original sources
- **Auto-Categorization**: Concept notes are automatically linked
- **Q&A Preservation**: Every query and answer is saved to wiki
- **Health Checks**: Auto-detects duplicates, orphan notes, and broken links
