## 六大核心功能

### 1. 市場研究自動化
- 內建 TypeBB 引擎（基於 OpenBB）
- 支援：股票、加密貨幣、外匯、宏觀經濟指標
- 技術分析：類 Excel 公式語法（SMA(close, 20)）
- RSS 新聞監聽：自動抓取、歸檔、分析

### 2. 多資產行情分析
- 支援：股票、加密貨幣、外匯、期權、期貨、ETF
- 內建技術指標：移動均線、RSI、布林帶
- 宏觀資料：美國勞工部 BLS 資料

### 3. 自主下單
- 支援 100+ 加密交易所（Binance、OKX、Bybit）
- 美股、股票期權期貨債券全覆盖
- **重要**：stage → commit → push 流程，不會自動亂買

### 4. 風控護盾
- 每帳戶獨立 Guard Pipeline
- 單筆最大倉位設定
- 交易冷卻時間
- 標的白名單

### 5. 上市公司研究
- 財務報表分析
- 分析師預期
- 內部人員交易記錄
- 財報日曆

### 6. 持續執行 + 記憶
- 持久化認知狀態
- 情緒狀態追蹤（影響決策風格）
- 跨會話工作記憶

---

## 三大設計原則

### 1. 檔案驅動
- 所有狀態存在本地 data/ 資料夾
- Markdown、JSON 配置、對話日誌
- 人類可讀，無資料庫，無雲端
- **資料主權完全在你手上**

### 2. 推理驅動
- 不靠規則下單，靠持續推理
- 每個決策背後有完整推理鏈
- 即時可見，不是黑箱

### 3. 作業系統原生
- 可呼叫 OS 能力：瀏覽器搜尋、iMessage 發訊息
- 可連線本地網路音箱叫醒你
- 可接本地大模型作為「大腦」

## 安全機制：三重保險

| 層面 | 說明 |
|------|------|
| **本地執行** | 所有狀態在本地，隨時能看到 Alice 在想什麼 |
| **stage → push 流程** | Git 模式，你不 commit/push，Alice 不能執行 |
| **Guard Pipeline** | 你設的規則，Alice 繞不過去 |
