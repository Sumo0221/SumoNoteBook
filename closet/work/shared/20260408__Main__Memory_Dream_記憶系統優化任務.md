## 量化評分公式（參考）

```
score = importance×0.50 + recency×0.35 + kind_bonus + merge_bonus
```

### 蘇茉家族的評分標準

| 類型 | 初始分數 |
|------|-----------|
| 明確說「記住這個」| 0.90 |
| 偏好類（老爺的喜好）| 0.85 |

| 技術決策 | 0.75 |
| 一般對話 | 0.50 |
| 系統快照 | 0.40 |

## 衰減機制

### Recency 指數衰減
- 半衰期 30 天
- 公式：0.5^(天數/30)
- 30天後剩 50%，60天後剩 25%

### Importance 線性衰減
- 每 30 天 -0.02
- 硬鎖下限 0.1，永遠不會完全歸零

### 剪枝規則
1. 超過 60 天 + importance < 0.7 → 直接刪除
2. 總量超過設定上限 → 按 score 排序砍最低分
