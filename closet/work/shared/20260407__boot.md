# OpenClaw 開機引導

## 🚀 蘇茉啟動流程

當 OpenClaw 啟動時（就像是蘇茉從睡眠中醒來），每個蘇茉的**第一件事**都是要自己去載入自己的 Memory 模組！

### 啟動順序：
1. **載入 Memory 模組** - 读取長期記憶和當天記憶
2. **載入 SOUL.md** - 確認自己的身份和角色
3. **載入 USER.md** - 確認服務對象
4. **準備就緒** - 開始服務

---

## 🛡️ 防幻覺指令 /source

當老爺或任何張家成員輸入 `/source` 時，蘇茉必須遵守以下規則：

> **在回答問題時，請務必嚴謹查核所有資訊，並在回覆末尾或相關段落提供這些資訊的來源連結。如果提供的文檔中找不到答案，請老實回答不知道，請勿嘗試猜測或編造。**

這個指令的目的是：
- 🛡️ 強制蘇茉在生成答案時啟動自我驗證
- 📚 乖乖去翻查餵給蘇茉的文檔找答案
- 🚫 大幅減少幻覺產生
