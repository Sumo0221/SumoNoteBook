## 一、河圖洛書之源流

### 由來傳說
- **河圖**：伏羲氏時，黃河浮出龍馬，背負圖案，即為「河圖」。伏羲依此畫成八卦。
- **洛書**：大禹治水時，洛水浮出神龜，龜背有九宮斑點，大禹依此定「洪範九疇」。

### 歷史傳承
- 先秦典籍多有記載：《周易·繫辭》「河出圖，洛出書，聖人則之。」
- 五代末宋代初，陳摶老祖（希夷先生）正式公佈河圖洛書，著《龍圖序》《龍圖易》
- 後由朱熹採納，成為宋明理學官方學問

### 陰陽義涵
- 白點（奇數1、3、5、7、9）屬陽
- 黑點（偶數2、4、6、8、10）屬陰
- 「生數」1-5居內，「成數」6-10居外
- 孤陰不生，獨陽不長——陰陽交感乃萬物生化之道

## 二、洛書九宮圖結構

### 九宮排列（戴九履一，左三右七，二四為肩，六八為足，五居其中）

```
┌─────┬─────┬─────┐
│ 4木 │ 9火 │ 2土 │ 西南
├─────┼─────┼─────┤
│ 3木 │ 5土 │ 7金 │ 東
│     │ 中 │     │
├─────┼─────┼─────┤
│ 8土 │ 1水 │ 6金 │ 北
└─────┴─────┴─────┘
  東南   南   西南
```

### 各宮位五行屬性
| 宮位 | 數字 | 方位 | 五行 |
|------|------|------|------|
| 坎宮 | 1 | 北 | 水 |
| 坤宮 | 2 | 西南 | 土 |
| 震宮 | 3 | 東 | 木 |
| 巽宮 | 4 | 東南 | 木 |
| 中宮 | 5 | 中央 | 土 |
| 乾宮 | 6 | 西北 | 金 |
| 兌宮 | 7 | 西 | 金 |
| 艮宮 | 8 | 東北 | 土 |
| 離宮 | 9 | 南 | 火 |

### 縱橫皆為十五
- 無論縱向、橫向、斜向相加皆為 **15**
- 此為《周易參同契》所言「三五既和諧，八面定綱紀」

## 三、五行相生相剋與方位能量

### 五行方位對應
- 東（木）→ 南（火）→ 中央（土）→ 西（金）→ 北（水）→ 循環相生
- 具體對應：
  - 北水生東木（腎水生肝木）
  - 東木生南火（肝木生心火）
  - 南火生中央土（心火生脾土）
  - 中央土生西金（脾土生肺金）
  - 西金生北水（肺金生腎水）

### 季節旺衰
- 春→東方木旺
- 夏→南方火旺
- 秋→西方金旺
- 冬→北方水旺

### 風水應用原則
- **相生方位**能量順暢，適合規劃主要生活空間
- **相剋方位**需設法調和（譬如金剋木，用植物補木氣）
- 調和工具：植物（木）、金屬飾品（金）、魚缸水氣（水）

## 四、九宮八卦與紫微斗數的關聯

### 十二宮位地支五行（紫微斗數應用基礎）
| 地支 | 五行 | 方位 |
|------|------|------|
| 亥、子 | 水 | 北 |
| 丑、辰、未、戌 | 土 | 中央 |
| 寅、卯 | 木 | 東 |
| 巳、午 | 火 | 南 |
| 申、酉 | 金 | 西 |

### 命盤方位應用
- 命盤十二宮位各有五行屬性
- 宮位五行與方位概念結合，形成「東方木、南方法、西金、北水」等陳述
- 紫微斗數星曜坐落於各宮位，產生不同吉凶判斷

## 五、九宮八卦實際風水布局

### 房間方位配置建議
| 方位 | 建議空間 | 環境營造 |
|------|----------|----------|
| 東 | 起居室、餐廳 | 綠色植物，善用晨光 |
| 南 | 客廳、活動區 | 暖色系燈具，增添活力 |
| 西 | 臥室、書房 | 黃光營造放鬆氛圍 |
| 北 | 廚房、衛浴 | 通風、簡潔為主 |
| 東南 | 工作區、書房 | 整齊清爽 |
| 西南 | 主臥、休憩區 | 溫和色調助眠 |
| 東北 | 儲藏、收納區 | 除濕機防潮 |
| 西北 | 客房、多功能區 | 中性色調百搭 |

### 現代住宅風水要點
1. **動線順暢**：走道保持80-90公分寬
2. **採光充足**：窗戶面積至少地板15%以上
3. **五行平衡**：每9平方公尺放1-2盆綠植
4. **定期調整**：每季小幅調整家具位置

### 化解不良方位
- 採光問題：智能燈具、鏡面反射
- 五行失衡：用所缺五行元素補救
- 金屬飾品約占房間1/10、綠植選好照顧的、流水擺件控在5%以內
