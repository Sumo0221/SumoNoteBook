## 🎬 實際內容

介紹如何使用 Grok3 和 Nano Banana 2 免費生成圖片和影片：

### 網站功能
| 功能 | 說明 |
|------|------|
| Grok3 | 免費、無限使用（不是最新版本）|
| Nano Banana Pro | 免費圖片生成 |
| Nano Banana 2 | 免費圖片生成 |
| 積分系統 | 10 積分（可繼續累積）|

### 操作流程
1. 登入網站（Google 或 Email）
2. 切換中文介面
3. 生成圖片：選擇 Nano Banana 2 模型 → 輸入提示詞 → 選擇解析度 → 生成
4. 生成影片：選擇 Grok3 模型 → 上傳參考圖 → 輸入提示詞 → 選擇解析度和時長 → 生成
