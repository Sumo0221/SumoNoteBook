## 🌿 2026-04-05｜學習主題：易經卜卦 — 六爻占卜基礎

### 一、六爻占卜概述

六爻占卜，又稱金錢卦、文王聖卦，相傳為周文王所創。
以三枚銅錢（或硬幣）搖六次，產生六個陰陽爻，由下而上排列，
構成一完整的六爻卦象。

---

### 二、起卦方法——銅錢搖卦法

**準備工具**：三枚相同的銅錢（乾隆通寶為佳）

**步驟**：
1. 將三枚銅錢合扣於雙掌之中
2. 靜心、誠心、意念集中，默念要測之事
3. 輕擲於地，反覆六次
4. 記錄時**由下往上**記（即由初爻到上爻）

**陰陽判定**：
- 銅錢有字的一面為陽，無字的一面為陰

| 結果 | 陰陽屬性 | 記號 |
|------|----------|------|
| 兩陽一陰 | 少陽 | ⚍（陽爻） |
| 兩陰一陽 | 少陰 | ⚋（陰爻） |
| 三陽 | 老陽 | ☰（陽爻，變陰） |
| 三陰 | 老陰 | ☷（陰爻，變陽） |

**老陽、老陰為動爻**：產生的新卦為**變卦**（之卦）

### 四、六十四卦構成

六十四卦由八卦**兩兩相疊**而成，分為八宮：
- 乾宮、坤宮、震宮、巽宮、坎宮、離宮、艮宮、兌宮
- 每宮領頭八個卦

**上下經之分**：
- 上經（三十卦）：以天道為主，如乾、坤、屯、蒙
- 下經（三十四卦）：以人事為主，如咸、恆、晉、明夷

**代表性卦象**：
- **乾為天**：純陽剛健，「潛龍勿用」「飛龍在天」
- **坤為地**：純陰柔順，「地勢坤，君子以厚德載物」
- **水雷屯**：萬物初生，創業艱難之象
- **地天泰**：陰陽交泰，萬事亨通
- **天地否**：陰陽阻塞，事事不順

---

### 五、世爻與應爻

- **世爻**：代表提問者自身之事（測自身以世爻為用神）
- **應爻**：代表對方或不相干之人（應爻為用神）

六爻卦中，世爻與應爻遙遙相對，是判斷吉凶的重要依據。

---

### 六、用神（六親）取用

六爻以六親配五行，取決於具體所測之事：

| 測項 | 用神 |
|------|------|
| 長輩身體 | 父母爻 |
| 感情（男測） | 妻財爻 |
| 貓走失 | 子孫爻 |
| 自身 | 世爻 |
| 對方 | 應爻 |

---

### 七、學習心得

陰陽五行為理論基礎，世應六親為判斷框架。

## 2026-04-05（晚上）學習主題：姓名學——五格剖象法

### 一、姓名學概述

姓名學是中國傳統命理學的重要分支，透過姓名的筆劃數理與陰陽五行來分析對人生的影響。古人認為姓名不僅是符號，更具有「靈動力」，透過呼喚名字，會對人的命運產生作用。

核心理念：「一命、二運、三風水、四積德、五讀書、六姓名」

姓名學認為，除了傳統算命以「知天命」為主之外，人類尚可經由其他方式來改變既有運勢，而姓名正是其中一個重要的「後天運」改造工具。

三大流派差異：
1. 重視筆劃數之門派（日本熊崎健翁系）：以81數理吉凶為主
2. 重視五行生剋之門派：以五格五行相生剋與天運五行為主
3. 重視納音與字形含義之門派：以字的發音五行與字形為主

---

### 二、姓名學歷史源流

| 時期 | 人物/事件 | 貢獻 |
|------|----------|------|
| 宋代 | 邵康節《皇極經世術》 | 論五音、四聲、字韻、數律 |
| 宋代 | 蔡九峰《八十一名數圖》 | 首創筆劃數之吉凶體系 |
| 明代 | 萬育吾《三命通會》 | 《五音看命法》三百字簡述 |
| 1928年 | 日本熊崎健翁 | 依蔡九峰著作創《熊崎姓名學》，授徒萬人 |
| 1936年 | 白惠文（台灣留學生） | 翻譯為漢文《姓名之命運學》傳入華人世界 |

重要結論：宋代之後，姓名學幾乎失傳中止千年，直至1928年日本熊崎健翁重新發揚光大，再由白惠文於1936年傳回華人世界。

---

### 三、康熙字典筆劃算法（核心基礎！）

姓名學筆劃數以《康熙字典》的部首筆劃數為準，非現代手寫筆劃數。

#### 特殊部首筆劃表

| 部首 | 名稱 | 計算法 | 舉例 |
|------|------|--------|------|
| 扌 | 提手旁 | 以「手」字計 = 4劃 | 提、撥、振 |
| 忄 | 豎心旁 | 以「心」字計 = 4劃 | 快、悟、情 |
| 氵 | 三點旁 | 以「水」字計 = 4劃 | 清、江、海 |
| 犭 | 犬字旁 | 以「犬」字計 = 4劃 | 狗、貓、獅 |
| 礻 | 半禮旁 | 以「示」字計 = 5劃 | 福、社、祖 |
| 王 | 斜玉旁 | 以「玉」字計 = 5劃 | 珍、珠、琴 |
| 艹 | 草字頭 | 以「艸」字計 = 6劃 | 花、芳、蓉 |
| 衤 | 衣字旁 | 以「衣」字計 = 6劃 | 初、袖、被 |
| 月 | 肉字旁 | 以「肉」字計 = 6劃 | 胖、肥、育 |
| 辶/辵 | 走馬旁 | 以「辵」字計 = 7劃 | 迎、運、遲 |
| 阝（邑）| 右耳旁 | 以「邑」字計 = 7劃 | 都、鄭、郊 |
| 阝（阜）| 左耳旁 | 以「阜」字計 = 8劃 | 陳、陽、院 |

計算原則：
1. 以康熙字典為準：所有筆劃必須依據《康熙字典》的繁體字計算
2. 以部首為準：特殊偏旁部首要按規定計算，非現代手寫筆劃
3. 部首算法為主，手寫算法為輔；若兩者批註皆凶，則應另取名字

### 四、五格剖象法（五格詳解）

| 五格 | 計算方式 | 象徵意義 | 備註 |
|------|---------|---------|------|
| 天格 | 姓筆劃 + 1（單姓）；姓筆劃相加（複姓）| 祖先運、先天運 | 不直接影響命運 |
| 人格 | 姓筆劃 + 名第一字筆劃 | 主運、核心人格 | 最重要的一格！ |
| 地格 | 名字所有字筆劃之和 | 前運（幼年至38歲）| 又稱「前運」 |
| 外格 | 名最後一字筆劃 + 1（單姓）| 副運、社交運 | |
| 總格 | 姓+名所有筆劃總和 | 後運（38歲後至晚年）| 又稱「晚境運」 |

天格計算：
- 單姓：姓的筆劃數 + 1（林=8+1=9；陳=16+1=17）
- 複姓：兩姓筆劃直接相加（歐陽=12+12=24；司馬=5+10=15）

人格計算：姓的筆劃 + 名字第一個字的筆劃
- 林木森：林（8）+ 木（4）= 12
- 人格象徵：核心性格、體質、人際關係基礎；天格→人格形成「成功運」

地格計算：名字所有字的筆劃之和（不含姓）
- 林木森：木（4）+ 森（12）= 16

外格計算：
- 單姓名：名最後一字筆劃 + 1（林木森：森12+1=13）
- 複姓名：第一個姓字的筆劃

總格計算：姓 + 名所有字的筆劃總和
- 林木森：林（8）+ 木（4）+ 森（12）= 24

### 五、三才配置——姓名學核心

天格、人格、地格三者合稱「三才」，是姓名學最核心的部分。

| 三才配置 | 五行關係 | 吉凶判斷 |
|---------|---------|---------|
| 天格 → 人格 | 祖先 → 自己 | 形成「成功運」：生助=吉；相剋=凶 |
| 人格 → 地格 | 自己 → 前運 | 形成「基礎運」：生助=吉；相剋=凶 |
| 天格 → 地格 | 祖先 → 前運 | 影響幼年環境 |

五行相生：木生火 → 火生土 → 土生金 → 金生水 → 水生木
五行相剋：金剋木 → 水剋火 → 木剋土 → 火剋金 → 土剋水

三才配置範例（「林木森」）：
- 天格 = 8+1=9（木）；人格 = 8+4=12（木）；地格 = 4+12=16（火）
- 天格木生人格木：木木相助，得長輩恩惠
- 人格木剋地格火：木生火為泄，可能體力耗費

### 七、姓名學與八字的配合

造運首要原則：姓名學皆主要以問者的生辰八字為改名造運之首要根據。

步驟：
1. 先了解先天八字命格：確認命局中的五行缺失
2. 姓名五行補正：根據八字選擇補足所缺的五行
3. 三才配置為核心：天格、人格、地格要形成良好配置
4. 數理要吉：人格與總格應為大吉或中吉數
5. 字音字形字義兼顧：名字要好聽、好看、有正面含義

重要警示：並非命中缺什麼就補什麼！五行太過也會成災，例如：總格屬水，其餘四格皆屬土，則「四土剋一水」格局，反而帶來更大凶險。

---

### 八、姓名學實用分析步驟

1. 確認姓名筆劃：以康熙字典為準，計算每個字的筆劃（注意特殊部首）
2. 計算五格：天格、人格、地格、外格、總格
3. 確認五行：依尾數法確認每格的五行
4. 分析三才配置：天格→人格、人格→地格是否相生助
5. 查詢數理吉凶：人格數、總格數是否為大吉/中吉
6. 綜合評判：結合八字命格，給出最終建議

### 九、學習心得

1. 五格剖象法是姓名學的主流方法，起源於宋代蔡九峰，發揚於日本熊崎健翁，以81數理和五行生剋為核心，五格之中，人格最為重要
2. 筆劃計算是姓名學的基本功，必須以《康熙字典》為準，特殊部首要牢記（扌=4、忄=4、氵=4等）
3. 三才配置決定姓名學的準確度：天格→人格形成「成功運」，人格→地格形成「基礎運」
4. 姓名學必須配合八字，姓名學是「後天運」的改造工具，必須先了解先天命格再取名/改名
5. 姓名學不能單獨決定命運，「好名字」需配合好風水，好日子，好積德，最重要的是積德行善
