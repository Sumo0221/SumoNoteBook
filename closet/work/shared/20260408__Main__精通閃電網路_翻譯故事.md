### 比特幣的支付問題

## 第十六章重點：閃電網路的隱私模型

| 特性 | 比特幣 | 閃電網路 |
|------|--------|----------|
| **交易** | 公開，隱私靠混淆 | 支付私密 |
| **地址** | 假名 | 通道公開 |
| **風險** | 交易公開可查 | 所有人都知道你跟誰連在一起 |

> 「閃電網路保護隱私」這句話最多對一半。另一半，很多人根本不知道自己暴露了什麼。

## 洋蔥路由原理

```
A ──→ C ──→ D ──→ B

- C 不知道 A 是起點
- D 不知道 B 是終點
- 每個節點只知道上一站和下一站
- 每一層只露出下一層的資訊
```
