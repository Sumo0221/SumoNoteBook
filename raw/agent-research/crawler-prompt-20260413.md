## 🎯 核心理念

> **一段 prompt = 爬蟲 + EPUB 生成**

用 Coding Agent（如 Claude Code）時，只需要給予一段完整的 prompt，就能讓 AI：
1. 分析目標網站結構
2. 爬取內容
3. 轉換為 EPUB 格式

---

## 💡 關鍵技巧

### 1. Prompt 結構

```markdown
我要幫老婆把小說抓到電子閱讀器，請幫我：
1. 分析這個部落格的結構
2. 爬取[小說名稱]所有章節
3. 轉換為 EPUB 格式
4. 確保目錄、書籤、元數據都正確

目標網站：[URL]
```

### 2. 必要資訊

| 資訊 | 說明 |
|------|------|
| **目標網址** | 小說所在的部落格/網站 |
| **小說名稱** | 要抓取的小說名稱 |
| **输出版本** | EPUB 格式 |

### 3. 自動化流程

```
User (一段 Prompt)
    ↓
Coding Agent
    ↓
┌─────────────────┐
│ 1. 分析網站結構 │
│ 2. 撰寫爬蟲代碼 │
│ 3. 執行爬蟲    │
│ 4. 轉換為 EPUB │
│ 5. 交付檔案    │
└─────────────────┘
```

## 🚀 蘇茉的應用：sumo_epub.py

根據這個技巧，蘇茉開發了 `sumo_epub.py`：

```bash

sumo_epub https://example.com/novel

# 功能
1. AI 分析網站結構
2. 爬取所有章節
3. 生成 EPUB 檔案
```
