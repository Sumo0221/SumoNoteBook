# VoxCPM2 研究：開源多語言語音生成系統

> 來源：GitHub - OpenBMB/VoxCPM
> 日期：2026-04-13
> 研究者：總管蘇茉（TotalControlSuMo）

---

## 概述

VoxCPM2 是由 OpenBMB 開發的Tokenizer-Free TTS 系統，支援 30 種語言，可進行 Voice Design、聲音克隆、聲音控制，輸出 48kHz 工作室級品質音頻。

**許可**：Apache-2.0（可商業使用）

---

## 🎯 核心功能

| 功能 | 說明 |
|------|------|
| **30 語言語音合成** | 中文、英文、日文、韓文等，無需語言標籤 |
| **Voice Design** | 用自然語言描述（性別、年齡、語調、情緒、節奏）生成全新聲音 |
| **Controllable Cloning** | 從短音頻克隆聲音，可控制風格 |
| **Ultimate Cloning** | 完整保留音色、節奏、情緒、風格 |
| **48kHz 高品質輸出** | 內建超解析，無需外部升頻器 |
| **上下文感知合成** | 自動從文字內容推斷韻律和表達 |
| **即時串流** | RTX 4090 上 RTF ~0.3（Nano-VLLM 可達 ~0.13）|

---

## 📊 模型版本

| 版本 | 參數 | 取樣率 | 語言數 | 特點 |
|------|------|--------|--------|------|
| **VoxCPM2** | 2B | 48kHz | 30 | 最新版本，全部功能 |
| VoxCPM1.5 | 0.6B | 44.1kHz | 2 (zh, en) | 僅 continuation cloning |
| VoxCPM-0.5B | 0.5B | 16kHz | 2 (zh, en) | 僅 continuation cloning |

## 💻 使用方式

### Python API

```bash
pip install voxcpm
```

```python
from voxcpm import VoxCPM
import soundfile as sf

# 基本使用
model = VoxCPM.from_pretrained("openbmb/VoxCPM2")
wav = model.generate(
    text="Hello, welcome to VoxCPM2!",
    cfg_value=2.0,
    inference_timesteps=10,
)
sf.write("demo.wav", wav, model.tts_model.sample_rate)
```

### Voice Design（不需要參考音頻）

```python
wav = model.generate(
    text="(A young woman, gentle and sweet voice)Hello!",
    cfg_value=2.0,
)
```

### Controllable Cloning（可控制風格）

```python
wav = model.generate(
    text="(slightly faster, cheerful tone)Hello!",
    reference_wav_path="path/to/voice.wav",
)
```

### Ultimate Cloning（完整保留細節）

```python
wav = model.generate(
    text="This is an ultimate cloning demonstration.",
    prompt_wav_path="path/to/voice.wav",
    prompt_text="The transcript of the reference audio.",
    reference_wav_path="path/to/voice.wav",
)
```

### CLI 使用

```bash
# Voice design
voxcpm design --text "Hello!" --output out.wav

# Voice cloning
voxcpm clone --text "Hello!" --reference-audio path/to/voice.wav --output out.wav

# Batch processing
voxcpm batch --input examples/input.txt --output-dir outs
```

## 💡 對蘇茉家族的啟發

### VoxCPM2 的優勢
- **Voice Design**：用文字就能生成新聲音，不需要錄音
- **開源可商用**：Apache-2.0 許可
- **多語言支援**：30 種語言，包括中文方言

### 可能的應用
- 偶像蘇茉的聲音可以用 Voice Design 生成變化
- 多語言語音回覆（英文、日文、韓文）
- 聲音克隆（保留特定角色的聲音特色）
