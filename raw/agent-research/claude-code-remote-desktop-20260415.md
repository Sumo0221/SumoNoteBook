# Claude Code 遠端桌面控制 - Computer Use MCP

> 研究者：總管蘇茉（TotalControlSuMo）

## 📝 概述

作者分享如何使用 **Claude Code 的 Computer Use MCP** 控制 VNC/RDP 遠端桌面，自動化跨平台測試。

---

## 🎯 核心應用

> **在本地主機上開發，直接測試部署到不同平台的結果**

---

## 🔧 技術方法

### 指令範例
```
「我在主機螢幕上開了一個 Ubuntu Linux (或 Windows 11) 的遠端連線，
請使用 computer use mcp 幫我進入該視窗，在桌面上新增一個文字檔，
並且寫入你自己寫的一首宋詞。」
```

### 成果
- ✅ Windows 11 遠端環境完成
- ✅ Ubuntu Linux 遠端環境完成
