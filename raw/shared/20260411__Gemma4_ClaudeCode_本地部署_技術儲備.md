# Gemma 4 + Claude Code 本地部署 - 技術儲備

## 📌 來源

YouTube: 「只有 6GB！Google Gemma 4 接入 Claude Code 深度实测」
頻道: YAHA学堂
URL: https://www.youtube.com/watch?v=wCZjRuspkIs

## 🔑 核心概念

用 Google Gemma 4（免費開源模型）取代昂貴的 Claude Code API

## ⚙️ 技術需求

| 項目 | 需求 |
|------|------|
| **RAM** | 16GB（Mac） |
| **模型** | Google Gemma 4（6GB） |
| **工具** | Ollama（本地跑模型） + Claude Code |
| **成本** | 免費（不用 API 費用） |
| **速度** | 實測比一般設定快 4-10 倍 |

## 📝 實作方式

1. 安裝 Ollama
2. 下載 Gemma 4 模型（6GB）
3. 設定 Claude Code 使用本地 Gemma 4
4. 從零做一個 Chrome 小恐龍遊戲

## ⚠️ 目前限制

- **沒有 16GB 硬體**：暫時無法實測
- 需要等未來有足夠記憶體的機器

## 🎯 何時可以重新考慮

- 當有 16GB+ RAM 的机器時
- 當 API 費用成為瓶頸時
- 當需要省錢跑本地模型時
