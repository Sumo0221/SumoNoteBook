## 二、刑事案件要點

### 2.1 主要罪名的法定刑

| 罪名 | 法條 | 法定刑 |
|------|------|--------|
| 過失致死 | 刑法第276條 | 5年以下有期徒刑、拘役或50萬元以下罰金 |
| 過失傷害 | 刑法第284條前段 | 1年以下有期徒刑、拘役或10萬元以下罰金 |
| 過失重傷害 | 刑法第284條後段 | 3年以下有期徒刑、拘役或30萬元以下罰金 |

### 2.2 自首（刑法第62條前段）

在「犯罪偵查機關未發覺犯人姓名」前，向警察或司法機關承認犯罪並表示願受裁判者，得減輕其刑。

**實務判斷標準（最高法院113年度台上字第4607號參照）：**
- 被告須於有偵查犯罪權限之人「知悉犯人姓名之前」即主動告知
- 須出於內心真誠悔悟，而非預期獲邀減刑
- 須向到場處理員警或偵查機關為之

### 2.3 緩刑（刑法第74條）

被告未曾因故意犯罪受有期徒刑以上刑之宣告，且認暫不執行為適當者，得宣告緩刑。

**常見緩刑條件（實務操作）：**
- 與被害人或家屬達成調解
- 支付一定金額之賠償（常見数十萬至數百萬元）
- 按时履行和解條件

**緩刑附帶負擔（刑法第74條第2項第3款）：**
- 法院得命犯罪行為人向被害人支付相當數額之財產或非財產上之損害賠償
- 此部分得為民事強制執行名義
- 未履行緩刑負擔且情節重大者，檢察官得聲請撤銷緩刑（刑法第75條之1第1項第4款）

### 2.4 實務常見過失情節

| 過失類型 | 說明 |
|---------|------|
| 未注意車前狀況 | 行經路口、變換車道時未注意前方路況 |
| 未保持安全距離 | 後車未保持可煞停之距離（道交規則第94條） |
| 轉彎未注意其他車輛 | 未依規定讓行、占用來車道搶先左轉 |
| 闖紅燈/紅燈右轉 | 違反交通號誌 |
| 開車門不注意 | 停車開啟車門時未注意其他車輛 |
| 行人穿越道未禮讓 | 行人穿越道前未停車禮讓行人 |

## 三、民事案件要點

### 3.1 實體法依據

| 法條 | 內容 |
|------|------|
| 民法第184條第1項前段 | 侵權行為一般條款：故意或過失不法侵害他人之權利者，負損害賠償責任 |
| 民法第191條之2 | 動力車輛：汽車、機車等動力車輛在使用中加損害於他人者，駕駛人應賠償 |
| 民法第193條第1項 | 侵害身體或健康者，對於被害人因此喪失或減少勞動能力或增加生活上之需要時，應負賠償責任 |
| 民法第195條第1項前段 | 侵害身體、健康、名譽等情節重大者，被害人得請求精神慰撫金 |
| 民法第217條 | 過失相抵：被害人與有過失者，法院得減輕或免除賠償金額 |
| 民法第229條、第233條 | 遲延利息：無確定期限之債，經催告未給付者，自催告時起負遲延責任；金錢債務遲延以週年利率5%計算 |
| 道路交通管理處罰條例第94條第1項、第3項 | 汽車同一車道行駛時，後車與前車應保持隨時可以煞停之距離；行駛時駕駛人應注意車前狀況 |

### 3.2 強制汽車責任保險理賠

**強制汽車責任保險法第32條：**
- 保險人已給付被害人之保險金，視為被保險人損害賠償金額之一部分
- 被保險人受賠償請求時，得扣除之

**實務計算方式：**
```
應賠償金額 − 強制險已理賠金額 = 被告應給付金額
```

### 3.3 親屬看護費用

**最高法院94年度台上字第1543號判決：**
- 親屬代為照顧被害人之起居，雖基於親情無需支付對價
- 但親屬付出之勞力並非不能評價為金錢
- 此種基於身分關係之恩惠，不能加惠於加害人
- 由親屬看護時，仍應認被害人受有相當於看護費之損害

### 3.4 過失相抵（民法第217條）

被害人与有过失者，法院得減輕賠償金額或免除之。

**實務常見比例分配：**
- 單一駕駛人全責：加害人100%
- 後車追撞前車：後車主要責任(80%)，前車如有疏失亦分擔(20%)
- 多車追撞：依各階段撞擊及個別過失程度分配

### 3.5 將來給付之訴

請求將來給付之訴，以有「預為請求之必要者」為限。
原告須證明：
- 未來支出之必要性
- 發生之確定性
（即必須合乎損害賠償填補實際所受損害之原則）

### 3.6 精神慰撫金

法院量定精神慰撫金應斟酌：
- 雙方之身分、地位、資力
- 加害之程度
- 被害人受有之痛苦程度
- 雙方經濟狀況

### 3.7 民事舉證與刑事差異

| 項目 | 刑事 | 民事 |
|------|------|------|
| 舉證原則 | 無罪推定、罪疑唯輕、嚴格證據主義 | 優勢證據原則 |
| 證明標準 | 超越合理可疑 | 高度蓋然性 |
| 拘束力 | 刑事認定之事實不當然拘束民事 | 民事法院仍應自行調查斟酌 |

## 四、代表性案例摘要

### 案例1：過失致死（臺灣臺北地方法院 115年度審交簡字第69號）

**案情：** 被告騎乘普通重型機車疏未注意車前狀況，撞擊正在推手推車之行人，致其後腦著地死亡。

**判決：** 
- 犯刑法第276條過失致死罪
- 有期徒刑6月，如易科罰金以1,000元折算1日
- 緩刑3年
- 應給付被害人家屬310萬元（含強制險）

**關鍵：** 自首(刑法第62條)減輕；調解成立、已部分履行為緩刑條件

---

### 案例2：民事損害賠償(交通)二審（臺灣高等法院 114年度簡上字第315號）

**案情：** 被告騎乘機車未保持安全距離，由後追撞原告，致原告左腿重傷骨折。

**相關法條：** 民法第184條、第191條之2、第193條、第195條、第217條；道路交通管理處罰條例第94條

---

### 案例3：民事損害賠償一審（臺北簡易庭 113年度北簡字第10586號）

**案情：** 三車連撞事故（原告撞訴外人機車 → 被告再追撞原告），原告左腿重傷。

**重要裁判意旨：**
- 民事舉證：優勢證據原則，原告僅需證明至「高度蓋然性」
- 強制險理賠可抵扣醫療費用
- 親屬看護可請求相當於看護費之損害
- 原告對自己傷害之發生亦與有過失(20%)，故被告僅須賠償80%
- 不能工作損失：原告須證明工作事實，退休者難以請求
- 將來給付須證明必要性與確定性

| 將來保養補品費用 | 9,479元 |

### 案例4：高等法院過失致死（臺灣高等法院 114年度交上訴字第76號）

**案情：** 被告小客車行經交岔路口，未注意已進入視線範圍之機車騎士，占用來車道搶先左轉，致被害人死亡。

**實務重點：**
- 左轉時未讓直行車先行，為常見過失類型
- 酒精濃度超標（0.16 mg/L）會加重量刑
- 未與被害人家屬達成和解會影響量刑
- 緩刑通常需以和解並履行為條件

## 五、相關法條彙整

### 刑事
- **刑法第276條**：過失致死 — 5年以下有期徒刑、拘役或50萬元以下罰金
- **刑法第284條**：過失傷害 — 1年以下；過失重傷害 — 3年以下
- **刑法第62條**：自首減輕
- **刑法第74條**：緩刑

### 民事
- **民法第184條**：侵權行為一般責任
- **民法第191條之2**：動力車輛責任（推定過失）
- **民法第193條**：人身侵害之財產賠償
- **民法第195條**：人身侵害之精神慰撫金
- **民法第217條**：過失相抵
- **民法第229條、第233條**：遲延利息（週年利率5%）
- **強制汽車責任保險法第32條**：強制險理賠抵扣

### 交通法規
- **道路交通管理處罰條例第94條**：保持安全距離、注意車前狀況

## 六、實務運作摘要

1. **車禍發生後第一步**：保留現場、報警、取得交通事故初步分析研判表
2. **刑事偵查**：警察機關製作筆錄、行車事故鑑定會鑑定
3. **調解**：車禍調解委員會或法院調解，可作為刑事緩刑依據
4. **強制險理賠**：醫療費用上限20萬元、身故200萬元（每人）
5. **民事求償項目**：醫療、復健、看護、工作損失、財物損失、慰撫金等
6. **舉證責任**：民事原告須負主要舉證責任，建議保存所有單據及診斷證明
