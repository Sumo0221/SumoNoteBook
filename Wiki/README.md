# Wiki 層 - 結構化知識庫

Wiki 層是 SumoNoteBook 知識組織的核心，專門用於存放經過整理的結構化知識。

## 存放內容

- **概念（concepts）**：經過整理的知識概念、定義、說明
- **摘要（summaries）**：長文檔的濃縮摘要、重點整理
- **MOC.md**：主題地圖（Map of Contents），連結各概念

## 用途

此層作為「百科全書」，是蘇茉家族的共享知識庫。所有蘇茉都可以：
- 查詢已整理的知識概念
- 閱讀重要資訊的摘要
- 透過 MOC 探索主題之間的關聯

## 與其他層的關係

```
Raw (原始資料) → Wiki (結構化知識)
                        ↑
                  Palace (記憶宮殿)
```

Wiki 是 Raw 到 Palace 的中繼站，將未經組織的原始資料轉化為可查找的結構化知識。
